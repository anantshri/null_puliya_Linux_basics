# VIM Basics

## Basic VIM Modes

* Command / Normal (default)
	Used to give commands to VIM and not edit text directly. 
* Edit
	This is true edit mode
* Visual Mode
	visual selection etc

## Command Mode options

1. :w : Write
1. :q : quit
1. :wq : Write and Quit
1. :!<cmd> : Execute Command
